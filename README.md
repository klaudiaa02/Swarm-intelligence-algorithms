# Swarm intelligence algorithms
PSO, BA, FA, CS

## Project Description
This repository contains implementations of selected **swarm intelligence algorithms** developed as part of an **engineering thesis (2024/2025)** focused on multidimensional optimization problems.

Implemented algorithms:
- PSO (Particle Swarm Optimization)
- BA (Bat Algorithm)
- CS (Cuckoo Search)
- FA (Firefly Algorithm)

The algorithms are used to optimize benchmark mathematical functions and compare their performance, convergence, and optimization efficiency.

---

## Benchmark Functions

The algorithms were tested on selected optimization functions:
- Rastrigin
- Rosenbrock
- Griewank
